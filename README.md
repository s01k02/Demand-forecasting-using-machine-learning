# 📊 Demand Forecasting using Machine Learning

## 📌 Project Overview
This project predicts retail store sales using historical data and machine learning techniques.  
The goal is to help businesses make better inventory and planning decisions.

---

## 🚀 Features
- 📈 Time Series Analysis
- 🔁 Lag Features (Previous sales)
- 📊 Rolling Mean (Trend capture)
- 🤖 XGBoost Model for prediction
- 🌐 Streamlit Web App (Interactive UI)
- 📉 Model Evaluation using MAE

---

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- XGBoost
- Streamlit
